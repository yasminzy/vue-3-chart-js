## Steps

1. Get the API key and put it in `/.env`.

```env
VITE_RAPIDAPI_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

2. Install the packages and run dev.

```bash
npm i
npm run dev
```
